PMEG Template Generator

Proof of Concept for Physician-Modified Endograft Template Generation

A web-based tool that allows surgeons to design fenestrations on vascular grafts and generate printable templates for precise modification during procedures.

ğŸ’¡ Concept

The software "unwraps" a cylindrical graft into a flat rectangle, preserving the precise positioning of fenestrations. Surgeons can:

Print the template on standard paper
Sterilize the template
Wrap it around the actual graft
Use holes as cutting guides
